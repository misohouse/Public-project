This project is rbs viewer for Windows telemetry log file.

This project was created using MFC in Visual Studio 2015. 

In this project, there is main algorithm in rbs_viewerDlg.cpp file.

To compile this project, You must change a project's property.

1st
C/C++ - General - Additional include Directories
: You should change a path of include folder in project.

2nd
Linker - General - Additional Library Directories
: You should change a path of include folder in project.
